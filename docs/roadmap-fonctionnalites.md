# Roadmap Fonctionnalités - Concert Chaussettes

## Analyse Concurrentielle

| Plateforme | Commission | Forces | Faiblesses |
|------------|-----------|--------|------------|
| **LiveTonight** | 15-20% | Curation française, qualité | Prix opaques, UX datée |
| **Gigstarter** | 12% | 50k+ artistes, Europe | UX complexe, moins de curation |
| **Linkaband** | Variable | IA matching, corporate | Orientation B2B, moins intime |
| **Sofar Sounds** | Revenue share | Expérience unique, communauté | Pas de booking privé |

| **Thumbtack** | Pay-per-lead | IA matching, mobile-first | Services généraux |

## Fonctionnalités à Implémenter

### Phase 1 - MVP+ (Priorité Haute)

#### Pour les Organisateurs
- [ ] **Système d'avis** - Notes 5 étoiles avec critères (qualité musicale, ponctualité, communication)
- [ ] **Favoris** - Sauvegarder des groupes pour plus tard
- [ ] **Multi-devis** - Demander des devis à plusieurs groupes simultanément
- [ ] **Messagerie intégrée** - Chat entre organisateur et groupe
- [ ] **Onboarding guidé** - Tutoriel première utilisation

#### Pour les Groupes
- [ ] **Calendrier de disponibilité** - Indiquer ses disponibilités
- [ ] **Statistiques profil** - Vues, contacts, taux de conversion
- [ ] **Galerie média améliorée** - Réorganisation photos/vidéos
- [ ] **Prix indicatifs** - Fourchette de tarifs affichable
- [ ] **Onboarding guidé** - Tutoriel complétion profil

### Phase 2 - Growth (Priorité Moyenne)

- [ ] **Matching IA** - Questionnaire → suggestions automatiques basées sur :
  - Type d'événement
  - Ambiance souhaitée
  - Budget
  - Localisation
  - Nombre d'invités
- [ ] **Paiement sécurisé** - Escrow avec Stripe
- [ ] **Génération de contrat** - Templates automatiques
- [ ] **Badges de confiance** :
  - Vérifié (identité confirmée)
  - Réponse rapide (< 24h)
  - Artiste recommandé (notes > 4.5)
  - Super hôte (pour organisateurs)
- [ ] **Notifications push** - Nouveaux messages, demandes de devis

### Phase 3 - Différenciation (Priorité Basse)

- [ ] **Packages expérience** :
  - "Apéro Jazz" - Jazz trio + suggestions vin
  - "Soirée Chanson" - Chanteur français + ambiance
  - "Brunch Musical" - Duo acoustique + playlist
  - "Mariage Intime" - Bundle cérémonie + vin d'honneur + soirée
- [ ] **"Coup de Cœur"** - Artistes sélectionnés par l'équipe
- [ ] **Application mobile PWA**
- [ ] **Analytics avancées** pour groupes (heatmap profil, etc.)
- [ ] **Programme de fidélité** pour organisateurs réguliers

## Positionnement

> **"La plateforme premium pour concerts privés intimes en France"**

### Proposition de valeur

**Pour les Organisateurs :**
> "Trouvez l'artiste parfait pour créer un moment inoubliable chez vous"

**Pour les Groupes :**
> "Jouez dans des lieux uniques, connectez avec votre public en toute intimité"

### Avantages compétitifs
- Commission compétitive (10-12%)
- Focus concerts intimes (< 100 personnes)
- Expertise française (chanson, jazz manouche, variété)
- Transparence totale des prix
- UX moderne et guidée
